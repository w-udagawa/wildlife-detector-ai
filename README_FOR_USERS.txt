■ はじめにお読みください

1. このフォルダをデスクトップに展開済みですか？
   まだの場合は、デスクトップに展開してください。

2. インストール方法
   「setup_for_users.bat」をダブルクリックしてください。
   
   ※「Windowsによって保護されました」と表示されたら
     「詳細情報」→「実行」の順にクリック

3. 詳しい使い方
   docs フォルダ内の「START_GUIDE_JP.md」を
   ご覧ください。
